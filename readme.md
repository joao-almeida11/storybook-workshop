# Storybook Workshop

## First Steps

Make sure you have Node.JS installed.

The first thing is to install the dependencies.

```sh
npm ci
```

To start the application in development mode

```sh
npm run dev
```

To start Storybook

```sh
npm run storybook
```

See `package.json` for more useful scripts.
